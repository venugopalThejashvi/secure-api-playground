## Run Locally

### Prerequisites
- **Java 17** (JDK)
- **Maven 3.8+**
- **Git**

### Steps
* clone the repo
* cd predict
* mvn spring-boot:run

### CONCEPTS
* Springboot basic crud
* jwt authentication
* env
* exception handler
* health check with/without JWT
