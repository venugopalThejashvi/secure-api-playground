## Run Locally

### Prerequisites
- **Java 17** (JDK)
- **Maven 3.8+**
- **Git**
- **Redis Server(Must be running locally)**

### Steps
* clone the repo
* cd predict
* mvn spring-boot:run

### CONCEPTS
* Rate Limiting with redis
* Springboot basic crud
* jwt authentication
* env
* exception handler
* health check with/without JWT
